# GDG Hack Frontend

A **game-based learning platform** built with **Next.js, TypeScript, and Tailwind CSS**, designed to help users learn **frontend development from scratch** in an interactive and engaging way. The platform combines structured lessons, multiplayer interaction, progression systems, bosses, and in-game items to turn learning into a playable experience.
